My Journey to My First Docker Container

This page documents my step-by-step journey of learning Docker by building a real Python + Flask application in a progressive, hands-on manner.

Project Initialization

Started with a clean and scalable project structure. Created separate folders for application code, templates, static assets, and Docker configuration.

Flask Application Setup

Built a simple Flask application with routing, template inheritance, and health check endpoints to ensure the app responds correctly.

Dockerization

Created a Dockerfile using an official Python base image. Installed dependencies, copied application code, and exposed the required port to run the Flask app inside a container.

Scaling with Templates & Layout

Introduced a base layout with reusable header and footer. Added multiple pages such as Home, Health, and Journey using template inheritance.

UI Enhancement

Enhanced the user interface using Tailwind CSS and Font Awesome. Focused on clarity, responsiveness, and clean design rather than unnecessary complexity.

Result & Learnings

Successfully built and ran my first Docker container. Gained a clear understanding of images, containers, Dockerfiles, and real-world development workflows.
